# CONTOH DATA EXCEL UNTUK IMPORT

Copy data ini ke Excel, lalu save dan import ke sistem:

## Format CSV (Copy paste ke Excel):

```
tanggal,tipe,kategori,uraian,nominal,keterangan
2025-01-05,pemasukan,Operasional,Saldo awal kas Januari,50000000,Dana awal bulan
2025-01-10,pengeluaran,Konsumsi,Belanja kebutuhan bulanan,500000,Belanja di toko
2025-01-15,pemasukan,Donasi,Donasi dari Bapak Ahmad,2000000,Donasi operasional
2025-01-20,pengeluaran,Transport,BBM kendaraan operasional,300000,Isi bensin
2025-01-25,pengeluaran,Utilitas,Bayar listrik dan air,450000,PLN dan PDAM
2025-02-01,pemasukan,Operasional,Saldo awal Februari,51250000,Carry over dari Januari
2025-02-10,pengeluaran,Gaji,Gaji karyawan bulan Februari,5000000,Gaji 3 karyawan
2025-02-15,pemasukan,Donasi,Donasi dari Ibu Siti,1500000,Untuk operasional
2025-02-20,pengeluaran,Konsumsi,Belanja kebutuhan,600000,Belanja bulanan
2025-03-01,pemasukan,Operasional,Saldo awal Maret,47150000,Carry over dari Februari
```

## Cara Pakai:

1. Buka Excel / Google Sheets
2. Copy semua baris di atas (termasuk header)
3. Paste ke Excel mulai dari cell A1
4. Save as Excel (.xlsx)
5. Import ke sistem

## Tips:
- Tanggal format: YYYY-MM-DD (contoh: 2025-01-05)
- Tipe: "pemasukan" atau "pengeluaran"
- Nominal: Angka saja tanpa Rp atau titik
- Keterangan: Opsional, bisa dikosongkan
